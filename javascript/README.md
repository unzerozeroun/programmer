# JavaScript

Tout code JavaScript peut être directement inclus à un document HTML entre les balises ```<script>...</script>``` ou dans un fichier externe dont l'extension est **.js** En ce dernier cas, le code peut alors être exécuté via Node.js côté serveur, ou par appel côté client via ```<script src="chemin_relatif.js"></script> ```
